3. 메시지 삽입 (iMessage)

▾
말풍선/프로필은 여기 박스 안에서만 내용을 수정해. 우측 미리보기에서 각 블록 좌상단에 나타나는 ⠿ 를 꾹 눌러 드래그하면 본문과의 위/아래 순서를 바꿀 수 있어.

4. HTML 삽입

▾
코드마다 독립된 박스로 관리되고, 추가하는 즉시 우측 미리보기에 블록으로 나타나. 메시지 블록처럼 미리보기에서 ⠿ 를 꾹 눌러 드래그하면 코드 박스 단위로 위/아래 순서를 바꿀 수 있어.
